#Requirements

- NodeJs 
- Npm
- Gulp

#Install 

Install dependencies

```bash
$ npm install
```

#Develop

Run gulp task serve
```bash
$ gulp serve
``` 

#Build 

Run gulp task build
```bash
$ gulp build
```
